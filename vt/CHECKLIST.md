# VT Coverage Checklist

The VT suite is maintained directly in YAML under `vt/specs/`.
Current target: one broader spec per page, with responsive coverage folded into the owning page spec where possible.

## Projects And Shell

- Projects page responsive states on wide, laptop, tablet, and mobile viewports in one spec.
- Project overview page with edit submit coverage.
- About page route coverage.
- Versions page empty, create, and edit coverage.
- Scenes page map creation, preview, scene editor, and sections overview coverage.

## Images

- One merged images workflow covering seeded detail/preview/edit, explorer keyboard selection auto-scroll, explorer drag/drop, empty search, upload, uploaded preview, rename, and filtered search.

## Asset Pages

- Characters page create/detail/edit/search coverage.
- Character sprites page upload/detail/preview/edit/search coverage.
- Sounds page upload/detail/audio preview/edit/search coverage.
- Videos page upload/detail/video preview/edit/search coverage.
- Transforms page detail/preview/edit/search coverage.
- Animations page detail/edit/search and editor route coverage.
- Spritesheets page import/detail/preview/search coverage.
- Particles page create/detail/edit/search coverage.

## UI Resource Pages

- Colors page detail/preview/edit/search coverage.
- Fonts page detail/glyph preview/edit/search coverage.
- Text styles page detail/dialog/search coverage.
- Layouts page detail/edit/search and editor route coverage.
- Controls page detail and keyboard action coverage.

## Variables

- One merged variables workflow covering empty state, string variable create/detail/edit/search, number variable create/detail/edit, and boolean variable create/detail/edit.
