# Windows Player Template

Place the release-built Windows player template here as:

```text
RouteVNPlayerTemplate.exe
```

Creator resolves it at runtime as:

```text
player-templates/windows/RouteVNPlayerTemplate.exe
```

The template is produced from `crates/routevn-packager/tauri-shell` and bundled
into Creator release builds. It is intentionally not generated by end users.

Build it with:

```bash
bun run player-template:build:win
```

Or from a Linux host with Docker:

```bash
bun run player-template:build:win:docker
```

The native build host needs:

- Rust target `x86_64-pc-windows-msvc`
- `cargo-xwin`
- LLVM/Clang with `llvm-rc` and `clang-cl` on `PATH`

The Docker build host only needs Docker. The image installs the pinned Rust
toolchain, `cargo-xwin`, Node, Bun, LLVM, and Clang.

The GitHub Actions workflow `windows-player-template.yaml` builds and uploads
the template artifact for release use.
